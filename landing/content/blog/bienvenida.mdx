# Bienvenidos a chandelierp

Estamos emocionados de presentar **chandelierp**, el sistema ERP diseñado específicamente para pequeñas y medianas empresas de candelería en Colombia.

## ¿Por qué un ERP especializado?

Las candelerías tienen necesidades únicas:

- **Recetas de producción (BOM):** Cada vela tiene una fórmula con cera, fragancia, colorante y pabilo. Necesitas saber exactamente cuánto cuesta producir cada una.
- **Control de inventario dual:** Materias primas que se transforman en productos terminados.
- **Punto de venta ágil:** Tanto para venta directa en tienda como para pedidos al por mayor.

## Lo que incluye chandelierp

- Gestión completa de productos y recetas
- Inventario con alertas de stock bajo
- POS optimizado para candelería
- Facturación electrónica
- CRM para gestionar clientes
- Contabilidad integrada con PUC colombiano
- Dashboard con métricas en tiempo real
