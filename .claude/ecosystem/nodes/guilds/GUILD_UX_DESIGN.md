# Guild: UX/UI & Design System

> Horizontal guild ensuring visual consistency, usability, and accessibility across all modules.

---

## Identity

| Attribute | Value |
|-----------|-------|
| **Level** | L5 — Cross-Functional |
| **Type** | Guild |
| **Chair** | UX Designer (Lead) |

---

## Composition

| Role | Responsibility |
|------|---------------|
| **UX Designer (Lead)** | User research, flow design, usability validation |
| **UI Designer** | Visual components, design system, iconography, motion design |
| **UX Writer / Content** | Microcopy, user guides (guia_usuario), tooltips, system messages |

---

## Responsibilities

| Area | Detail |
|------|--------|
| **Design system** | Maintain tokens, components, patterns, and documentation |
| **User research** | Monthly interviews, session analysis, usability tests |
| **Prototyping** | Clickable prototypes for all major features before development |
| **Accessibility** | WCAG 2.1 AA compliance enforcement |
| **Content** | Consistent, human-friendly microcopy and documentation |
| **Cross-squad consistency** | Review every squad's UI output for design system compliance |

---

## Outputs

| Output | Destination |
|--------|------------|
| Design system updates | Frontend + All squads |
| Prototypes | Squads (before development) |
| Usability test reports | CPO + Head Product |
| User research synthesis | CPO |
| Content guidelines | All squads + Technical Writer |

---

## Metrics

| Metric | Target |
|--------|--------|
| Design system adoption | ≥ 95% |
| Usability test pass rate | ≥ 80% task completion |
| Accessibility compliance | 100% on core flows |
| User interviews / month | ≥ 4 |
| Content consistency score | ≥ 90% (audit) |

---

## Cadence

| Activity | Frequency |
|----------|-----------|
| Guild meeting | Bi-weekly |
| Design system review | Monthly |
| User interviews | Monthly (≥ 4) |
| Accessibility audit | Per release |
| Design critique session | Bi-weekly |
