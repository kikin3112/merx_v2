# Guild: Data & Analytics

> Horizontal guild owning business intelligence, product analytics, and ML/AI readiness.

---

## Identity

| Attribute | Value |
|-----------|-------|
| **Level** | L5 — Cross-Functional |
| **Type** | Guild |
| **Chair** | Data Analyst |
| **Scope** | BI + Product Analytics + ML/AI (progressive) |

---

## Composition

| Role | Responsibility |
|------|---------------|
| **Data Analyst** | Business metrics, dashboards, KPIs, reporting for ERP users |
| **BI Engineer** | ETL, data pipelines for accounting/sales/inventory reports |

---

## Responsibilities

| Area | Detail |
|------|--------|
| **BI & Reporting** | Dashboards for tenants (financial, sales, inventory KPIs) |
| **Product Analytics** | Usage metrics, feature adoption, funnel analysis (PostHog/Umami) |
| **Data pipelines** | ETL for cross-module reporting |
| **ML/AI Foundation** | Build data infrastructure for future ML features (Phase 4+) |
| **Metrics definition** | Define and validate North Star and squad-level metrics |

---

## Analytics Stack ($0)

| Layer | Tool | Cost |
|-------|------|------|
| **Product analytics** | PostHog (free tier) or Umami (self-hosted) | $0 |
| **BI dashboards** | Metabase (open-source) or Grafana | $0 |
| **ETL** | Custom Python scripts + PostgreSQL views | $0 |
| **ML (future)** | Scikit-learn, pandas (Python ecosystem) | $0 |

---

## Progressive Data Maturity

| Phase | Focus | Capabilities |
|-------|-------|-------------|
| **Phase 0-1** | Basic reporting | SQL views, simple dashboards, manual exports |
| **Phase 2** | Product analytics | Feature adoption tracking, funnel analysis, A/B testing |
| **Phase 3** | Advanced BI | Cross-module reports, custom tenant dashboards |
| **Phase 4+** | ML/AI | Demand forecasting, anomaly detection, smart suggestions |

---

## Outputs

| Output | Destination |
|--------|------------|

| Product analytics dashboards | CPO + VP Eng |

| ML/AI recommendations (future) | CPO + Squads |
