# Guild: Security (SecOps)

> Horizontal guild ensuring security is integrated into every stage of development.

---

## Identity

| Attribute | Value |
|-----------|-------|
| **Level** | L5 — Cross-Functional |
| **Type** | Guild |
| **Chair** | Security Engineer (Lead) |

---

## Composition

| Role | Responsibility |
|------|---------------|
| **Security Engineer (Lead)** | Security policy, audits, pen testing, incident response |
| **AppSec Engineer** | Maintains Semgrep, Gitleaks, Bandit rules. Security PR reviews |
| **Compliance Analyst** | Regulatory compliance (data protection, e-invoicing, DIAN standards) |

---

## Responsibilities

| Area | Detail |
|------|--------|
| **Security pipelines** | Maintain 6 CI/CD security gates |
| **Policy** | Define and update security policies |
| **Pen testing** | Quarterly pen tests using free tools |
| **Security champions** | Train and support security champions in each squad |
| **Compliance** | DIAN e-invoicing compliance, Habeas Data (Colombia) |
| **Incident response** | Lead security incident response process |
