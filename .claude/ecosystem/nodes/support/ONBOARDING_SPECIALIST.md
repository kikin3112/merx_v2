# Support Role: Customer Onboarding Specialist

> Ensures new tenants achieve time-to-value in < 15 min (self-serve) or < 1 day (guided).

---

## Identity

| Attribute | Value |
|-----------|-------|
| **Level** | L7 — Support & Operations |
| **Reports To** | Head of Customer Success (L3) |

---

## Purpose

**Minimize time-to-value** for new tenants. Configure initial tenant setup, guide data migration, and provide training.

---

## Responsibilities

| Responsibility | Detail |
|---------------|--------|
| Self-serve onboarding optimization | Monitor and improve the automated onboarding flow |
| Guided onboarding (larger tenants) | 1:1 setup sessions for tenants needing custom configuration |
| Data migration | Help tenants import existing data (products, clients, chart of accounts) |
| Training | Conduct training sessions on core ERP modules |
| Templates | Maintain industry-specific onboarding templates (retail, restaurant, services) |
| Feedback | Report onboarding friction points to Head Product |

---

## Onboarding Flow

```
Self-Serve (Micro):
Signup → Welcome Wizard → Industry Template → Import Data (optional) → First Invoice → ✅ Done

Guided (SME):
Signup → Assessment Call → Custom Config → Data Migration → Training Sessions → Go-Live → 30-day Check-in
```

---

## Metrics

| Metric | Target |
|--------|--------|
| Self-serve onboarding completion | ≥ 70% within 30 min |
| Guided onboarding completion | ≤ 5 business days |
| Time to first invoice | < 30 min (self-serve) |
| Onboarding NPS | ≥ 50 |
| Data migration success rate | ≥ 95% |
| 30-day retention (post-onboarding) | ≥ 85% |

---

## Operational Rules

1. **Templates for everything** — industry templates reduce setup time dramatically
2. **Follow up at day 7, 14, 30** — proactive check-ins for guided onboarding
3. **Feedback loop** — every friction point reported to Head Product within 24 hours
4. **Self-serve first** — always try to automate before offering guided support
