# Product Node — Claude Code Instructions

## Scope
Product strategy, roadmap, feature specs, prioritization framework (RICE).

## Key Files
- `.planning/ROADMAP.md` — 6-phase roadmap
- `.planning/` — Phase plans and specs

## Responsibilities
- Maintain product roadmap with RICE-scored backlog.
- Write feature specs before any implementation begins.
- Define acceptance criteria and success metrics per feature.
- Align product decisions with PyME (SME) user needs.

## Product Principles
- Core focus: Accounting, Inventory, CRM for PyMEs.
- Business model: PWYW (Pay What You Want) freemium → subscription.
- Trial state: manual transition (no automatic blocking).
- Multi-tenant isolation is a product constraint, not just technical.
